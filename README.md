# Netflix Data Analysis

This project explores Netflix’s catalog using Python and Pandas to uncover trends in movies and TV shows on the platform.

The analysis focuses on understanding patterns in content types, release years, genres, and global content production through exploratory data analysis.

## Tools Used
- Python
- Pandas
- Jupyter Notebook
- VS Code

## Project Goals
- Explore Netflix’s dataset structure
- Clean and analyze the data
- Identify trends in movies vs TV shows
- Analyze release year distribution
- Explore global content production

## Dataset
The dataset includes information about Netflix titles such as:
- Title
- Type (Movie or TV Show)
- Country
- Release year
- Rating
- Duration
- Genre categories
- Description

## Future Improvements
- Create visualizations of key trends
- Build interactive dashboards
- Explore recommendation patterns
